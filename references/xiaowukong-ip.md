# 小悟空 IP

## 角色定义

小悟空是本 skill 中文正文配图的固定视觉 IP，基于七龙珠幼年孙悟空形象。

默认每张图都要出现小悟空。小悟空不是吉祥物，不是贴纸，不是可爱装饰，而是正在认真参与系统运转的热血工作者。

> **角色基准**：参考 `assets/xiaowukong-three-view.png` 三视图，确保形象一致。

## 外形

- **发型**：标志性黑色刺猬头，发丝尖锐向上炸开，覆盖额头。
- **面部**：大眼睛、小鼻子、天真热血但认真的表情，脸颊圆润。
- **上身**：白色无袖背心（内衣/道服内衬风格）。
- **下身**：宽松蓝色灯笼裤，腰部系浅蓝色布带打结。
- **手腕**：红色护腕。
- **脚部**：深蓝色平底鞋，白色鞋底。
- **道具**：肩挎如意棒（红色长棍），从右肩斜跨至左腰。
- **头身比**：Q版 3~4 头身，符合幼年悟空形象。
- **轮廓**：手绘线条感，略带抖动，不机械不矢量。

## 性格

- 天真热血，但做事很认真。
- 像一个永远充满干劲的系统操作员/战士。
- 有点莽，但不蠢——先冲上去再说。
- 执着，不服输，遇到困难会变强。
- 不卖萌，但有一种少年特有的直率幽默。
- 像在白板草图里真的负责某个冒险任务。

## 常见职责

让小悟空承担核心动作：

- 挥舞如意棒劈开信息壁垒。
- 飞在空中搬运素材。
- 用龟派气功把杂乱信息轰成结构。
- 卡在断点里，用力掰开。
- 在机器里拉"判断"杆。
- 变成筛选漏斗，用武术分拣流量。
- 用如意棒挑起"素材鱼"。
- 盖章承接话术，动作干脆利落。
- 牵着承接路径，像牵着飞龙。
- 举警告牌看坑，表情严肃。
- 从洞里伸手但接不住内容，一脸不服。
- 在旁边搬砖、搭桥、开门、分拣、修炼、记录。

## 动作池

**继承自小黑**（通用操作类，适合搬运/分拣/组装/维护类隐喻）：
- 拉、扛、塞、捞、压、称、缝、剪、拧、守、推、接、拆、标记、回收。

**悟空专属**（热血战斗类，适合突破/冲击/修炼类隐喻）：
- 劈、踢、飞、举、修炼、变身、挥棒、冲击。

两组动作可自由混搭。动作要服务核心意思，不要为了热血而热血。

## 禁止

- 不要把小悟空画成过度可爱的吉祥物或儿童卡通角色。
- 不要给小悟空复杂服装、表情包、闪亮眼睛。
- 不要让小悟空只是站在角落里看。
- 不要让小悟空抢走结构表达。
- 不要把小悟空画得太商业、太圆润、太精致。
- 不要偏离标准形象：必须是黑刺猬头 + 白背心 + 蓝灯笼裤 + 红护腕。不能画成超级赛亚人（金发），不能画成成年悟空，不能换装。

## 判断标准

如果去掉小悟空，图的核心隐喻还能完全成立，说明小悟空太装饰了；要重写提示词，让小悟空成为动作主体。
